# 🎬 GoodCine - Sistema de Reserva de Assentos

Interface de um sistema de reserva de assentos de cinema, desenvolvida com HTML, CSS e JavaScript.

## 🚀 Funcionalidades
- Seleção de assentos interativa
- Interface visual de sala de cinema
- Feedback visual dos assentos selecionados

## 🛠️ Tecnologias
- HTML
- CSS
- JavaScript

## ⚙️ Backend (em desenvolvimento)
Este projeto possui integração planejada com:
- API em C#
- Banco de dados MySQL

A versão online representa apenas o front-end da aplicação.

## 🌐 Demo
https://rafael-santosr.github.io/cinema-seat-booking-system/

## 🎯 Objetivo
Projeto desenvolvido para praticar construção de interfaces, lógica de seleção e integração futura com backend.
